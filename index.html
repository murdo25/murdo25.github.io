Ben Murdoch

Research Scientist

I research autonomous agents that learn emergent languages in order to facilitate human robot interaction. While it’s possible to download and analyze large corpora full of symbolic information like Wikipedia or a knowledge graph, those symbols exist only as abstract entities disconnected from elementary sensory perception. The bleeding together of symbolic reasoning and the physical world is where language emerges. This illustrates one of the biggest flaws of twenty first century chatbots. How can we expect a chatbot to chat about a world it’s never seen? I propose we solve this problem through emergent language. If we want robots to understand language, they will need to evolve it. I use reinforcement learning as a method to establish learnt communication protocols across multi agent teams and by doing so create the next generation of social robots.
